# storybook-vue

## 模块职责

- 作为 `ui-public-vue` 第一阶段的展示、评审与视觉回归入口。
- 展示 preset、theme pack、light / dark mode 的切换结果。
- 作为设计验收面，不作为生产运行时真相 owner。
- 当前额外承接一条浏览器烟测入口，用于验证高风险公共组件在 Storybook iframe 中的真实交互路径。

## 明确不负责

- 不负责生产页面路由与业务功能。
- 不负责主题 token 真相。
- 不负责企业 `TDesign` 预设运行时。
