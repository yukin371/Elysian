# 交互重设计：分栏工作台

> 日期: 2026-05-08
> 状态: 设计确认
> 范围: example-vue 应用整体交互模式重设计

## 1. 设计目标

从"展示信息的后台"转变为"完成任务的工作台"。

核心原则：
- **操作链路最短**：点击行即展示详情，双击即编辑，无弹窗打断
- **信息密度最高**：同屏展示列表 + 详情/编辑，不需要来回切换
- **视觉层级清晰**：工具条 → 列表 → 面板，三层分明
- **删掉所有冗余**：说明卡片、统计摘要、重复的状态消息、步骤条

## 2. 壳层框架：三栏式工作台

```
┌─────────────────────────────────────────────────┐
│ 顶栏: 搜索 | 面包屑 | 用户头像 | 全局动作       │
├──────┬──────────────────────┬───────────────────┤
│      │                      │                   │
│ 侧栏  │   主工作区            │   上下文面板       │
│ 导航  │   (列表/表格)         │   (详情/编辑)      │
│      │                      │                   │
│ 模块树 │   查询工具条嵌入表头    │   选中即展示       │
│ 收藏   │   表格为主体         │   无选中时收起      │
│ 最近   │   操作在行内          │   编辑同位替换      │
│      │                      │                   │
│ 200px │   flex-1             │   360-480px       │
├──────┴──────────────────────┴───────────────────┤
│ 状态栏: 当前模块状态 | 记录数 | 选中状态           │
└─────────────────────────────────────────────────┘
```

### 侧栏 (200px)

- 模块树：层级导航，点击直接切换工作区
- 收藏：用户收藏的常用模块
- 最近：最近访问的工作区
- 折叠态：只显示图标，hover 展开标签

### 顶栏

- 全局搜索（跨模块搜索并直接跳转）
- 面包屑（显示当前位置层级，如：系统管理 > 用户管理）
- 用户头像 + 全局设置
- 不放模块级操作

### 状态栏 (底部薄条)

- 当前模块状态（在线/离线/无权限）
- 记录总数
- 选中记录信息
- 不占用主视觉

### 上下文面板

- 默认隐藏，选中记录后滑入
- 详情模式：360px（紧凑信息展示）
- 编辑模式：480px（需要表单空间）
- 关闭后列表自动占满宽度

## 3. 标准 CRUD 工作区

适用页面：用户、角色、菜单、部门、字典、岗位、系统配置、通知、租户

### 工具条

```
┌─────────────────────────────────────────┐
│ [+ 新增] [筛选▼] [更多▼]     🔍 搜索...  │
└─────────────────────────────────────────┘
```

- 一行完成所有工具操作
- 新增按钮主色强调
- 搜索框嵌入工具条，placeholder 提示可搜索字段
- 筛选下拉展开高级筛选条件
- 更多操作包含：导入、导出、批量删除等

### 表格

- 占满主区高度
- 行内操作放 `⋯` 菜单（编辑/删除/更多）
- 行选中高亮
- 底部紧凑分页：页码 + 前后翻页 + 总数

### 交互规则

- **点击行** → 右侧面板展示详情
- **双击行** → 右侧面板切换为编辑模式
- **删除** → 面板内显示危险区确认
- **无独立查询区**：搜索框就是筛选入口
- **无说明卡片/统计摘要块**
- **无弹窗**

## 4. 上下文面板

### 详情模式

```
┌───────────────────────┐
│ 用户详情          [×]  │
│                       │
│ ┌─────────────────┐   │
│ │ 头像  张三       │   │
│ │       admin@... │   │
│ │       [编辑] [×] │   │
│ └─────────────────┘   │
│                       │
│ 基本信息               │
│ 用户名   zhangsan      │
│ 邮箱     admin@...     │
│ 角色     管理员        │
│ 状态     ● 启用        │
│                       │
│ 组织信息               │
│ 部门     技术部        │
│ 岗位     高级工程师    │
│                       │
│ [编辑] [删除]          │
└───────────────────────┘
```

### 编辑模式

点击编辑或双击行后，同一面板内原地变为表单：

- 顶栏变为"编辑用户 + [保存][取消]"
- 表单字段与详情字段一一对应
- 底部固定保存/取消栏
- 保存后自动回到详情模式

### 设计规则

- **无选中时**：面板收起，列表占满宽度
- **选中后**：面板滑入 360px
- **编辑时**：面板宽度自动扩展到 480px
- **新建时**：面板直接打开为编辑表单模式
- **删除确认**：面板内显示危险区（红色底 + 确认/取消按钮）
- **面板内容分组**：可折叠 section，默认展开第一个

## 5. 复杂工作区：生成预览页

当前最复杂的页面（699 行），重设计为四块布局：

```
┌────────────────────────┬────────────────────────┐
│  ┌─ 配置面板 ────────┐ │  ┌─ 预览结果 ────────┐ │
│  │                    │ │  │                    │ │
│  │ 输入: [URL] [文件] │ │  │ 文件列表 (12)      │ │
│  │ Schema: [选择▼]   │ │  │ ┌──────────────┐  │ │
│  │ 目标: Vue ● React │ │  │ │ ▸ src/app.ts  │  │ │
│  │ 冲突: 跳过 ● 覆盖 │ │  │ │   src/util.ts │  │ │
│  │                    │ │  │ │ ▸ src/db.ts   │  │ │
│  │ [刷新预览]         │ │  │ └──────────────┘  │ │
│  └────────────────────┘ │  │                    │ │
│                        │  │ ┌─ 选中文件详情 ──┐ │ │
│  ┌─ 会话状态 ────────┐ │  │ │ src/app.ts      │ │ │
│  │ 状态: ● 已就绪     │ │  │ │ [差异] [应用]   │ │ │
│  │ [审核] [全部应用]  │ │  │ │ ──────────── │ │ │
│  └────────────────────┘ │  │ │ diff 内容...    │ │ │
│                        │  │ └──────────────┘  │ │
│                        │  └────────────────────┘ │
│     左栏 40%           │       右栏 60%          │
└────────────────────────┴────────────────────────┘
```

### 四个功能块

1. **配置面板（左上）**：输入方式、Schema、目标、冲突策略，一个"刷新预览"按钮
2. **会话状态（左下）**：状态指示 + 审核/应用操作，只出现一次
3. **文件列表（右上）**：预览结果的文件列表，带状态标记
4. **文件详情（右下）**：选中文件后展示 diff + 单文件操作

### 简化要点

- 删除所有说明文本和步骤条
- "刷新 → 审核 → 应用"构成唯一主节奏
- 会话状态只出现一次
- 文件操作在文件详情内，不单独做工具栏
- 不再用弹窗做二次确认

## 6. 特殊页面

### 树形管理页（菜单、部门）

- 主区改为左树右表
- 左侧展示树结构，可折叠/展开
- 右侧展示选中节点的子项列表
- 上下文面板规则不变

### 文件管理页

- 工具条增加上传按钮（主色强调）
- 支持拖拽上传到主区
- 文件预览在上下文面板中完成（图片缩略图 / 代码高亮）

### 操作日志页

- 时间轴筛选器嵌入工具条（今日/近7天/近30天/自定义）
- 日志条目用紧凑列表而非表格
- 上下文面板展示完整日志详情

### 在线会话页

- 会话列表为主体
- 状态用色点标记（绿=活跃，灰=过期，红=异常）
- 上下文面板展示会话详情 + 强制下线操作

## 7. 全局快捷键

| 快捷键 | 操作 |
|--------|------|
| `/` | 聚焦搜索框 |
| `Esc` | 关闭上下文面板 / 取消编辑 |
| `↑↓` | 表格中上下移动选中行 |
| `Enter` | 编辑选中行（等同双击） |
| `N` | 新建记录 |
| `Del` | 删除选中记录 |

## 8. 删除清单

以下元素从所有页面中移除：

- 独立查询条件区域（改为工具条内嵌搜索）
- 说明卡片/统计摘要块
- 模块状态警告消息（移到底部状态栏）
- 步骤条和流程说明文本
- 重复的会话/状态信息
- 弹窗式详情展示（改为面板）
- 弹窗式编辑表单（改为面板原地编辑）
- 弹窗式删除确认（改为面板内危险区）
- 冗余的分页控件（统一为紧凑底栏）
