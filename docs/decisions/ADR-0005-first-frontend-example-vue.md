# ADR-0005 First Frontend Example Uses Vue

状态：`accepted`

日期：`2026-04-20`

## 背景

平台长期方向是前端可插拔，同时支持 `Vue` 和 `React`。但当前仓库还没有真实前端示例，如果继续并列讨论两条路线，会拖慢首个垂直切片落地。

## 决策

1. 首个真实前端示例应用使用 `Vue`。
2. `React` 保持为后续适配目标，不在当前阶段并行实现完整示例。
3. `packages/frontend-vue` 作为首个适配层承载体，后续真实页面优先围绕它验证。

## 理由

- 项目最初的技术方向就是 `Elysia + Vue + Tailwind`，先落 `Vue` 可以减少决策成本。
- `Vue` 作为第一个示例足以验证平台的前端适配边界，并不否定后续 `React` 支持。
- 先有一个真实示例，比同时维护两套半成品更有价值。

## 影响

- 首个前端示例应用命名为 `apps/example-vue`
- 后续首个垂直切片的 UI 承载体默认是 Vue
- `React` 相关工作仍保留在架构层和占位包层，不进入当前实现主线

## 暂不决策项

- 未来 React 示例应用的落地时机
- 前端状态管理方案
- UI 组件库或设计系统方案
